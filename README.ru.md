**Все ваши панели Google Search Console в одном месте, плюс встроенный пакет ИИ-инструментов для SEO и приватная индексирующая сеть.**

## Почему OpenGSC

Сервисы вроде seogets.com просят **$19–$79/месяц** за визуализацию данных Google Search Console, которыми вы уже владеете. OpenGSC дает вам ту же ключевую аналитику — клики, показы, CTR, позиции, ключевые слова Striking Distance (на пороге топ-10), угасание контента (Content Decay) и каннибализацию — по цене VPS за $5/месяц, потому что это *и есть* ваш VPS: один скрипт установки, собственная база данных SQLite, собственный домен и собственное OAuth-приложение Google. Данные вашей Search Console никогда не проходят через сторонние серверы.

Помимо дашборда, OpenGSC содержит две вещи, которых нет в большинстве инструментов GSC: полноценный **пакет ИИ-инструментов для генерации контента** (`/seo-tools` — анализ конкурентов, генерация структуры на основе сущностей, написание статей целиком, GEO/AI-search аудит цитирования, отслеживание тональности упоминаний бренда) и **приватную сеть индексации** (`/indexer` — ваша собственная инфраструктура сайтов-дорвеев с клоакингом и верификацией ботов для быстрого и бесплатного индексирования).

| | OpenGSC | Типичный SaaS дашборд GSC |
|---|---|---|
| Цена | Бесплатно навсегда | $19–$79/месяц |
| Хостинг | Ваш собственный VPS | Их облако |
| Приватность данных | 100% приватность | Хранятся на их серверах |
| Сайты / Google аккаунты | Безлимитно | Ограничено тарифом |
| ИИ SEO-пакет контента | ✓ Включен, ваш API-ключ | Редко доступен |
| Приватная сеть индексации | ✓ Включена | Не предлагается |
| Открытый исходный код | ✓ | ✗ |
| Время настройки | ~5 минут, одна команда | Мгновенная регистрация |

**Честные компромиссы:** вам нужен VPS и домен (Google OAuth не работает на чистом IP-адресе), обновления вы ставите сами (`git pull && npm run build && pm2 restart`), и в системе нет встроенного разделения на команды или white-label. Если это приемлемый компромисс ради контроля над своими данными и отсутствия подписок — читайте дальше.

## ✨ Возможности

### Объединенный дашборд GSC

- **Один дашборд, все аккаунты** — любой сайт из любого подключенного Google-аккаунта на одном экране с графиком спарклайнов трафика для каждого ресурса.
- **Портфельные дашборды Google · Bing · Яндекс** — переключение всего дашборда между поисковиками в один клик. Google читается из локальной базы; **Bing** и **Яндекс** тянутся вживую из их Webmaster API по всем подключённым аккаунтам и показывают *их собственные* верифицированные сайты (а не только список Google), те же KPI-карточки, карточки сайтов со спарклайнами, сортировки, поиск, теги и CSV. Живые данные движков кэшируются на сервере — вкладка открывается мгновенно после первой сборки; кнопка **Sync** обновляет все три поисковика разом.
- **Быстрые периоды** — кнопки 7д / 28д / 3м / 6м / 12м / 16м, а также сравнение с предыдущим периодом, год к году или полностью кастомные диапазоны.
- **Интерактивные переключатели** — клики, показы, CTR и средняя позиция, с мгновенно пересчитываемой суммарной статистикой по всей сетке сайтов при фильтрации по тегам.
- **Быстрая ручная проверка индексации** — бейдж рядом с каждым сайтом открывает поиск `site:domain.com` в новой вкладке и следует активной вкладке движка (**G** → Google, **b** → Bing, **Я** → Яндекс).
- **Теги, избранное, сортировка и скрытие** — управляйте сотнями сайтов, закрепляйте важные проекты и скрывайте неактивные.
- **Режим приватности (Blur)** — в один клик размывает названия аккаунтов, email, домены и метрики для безопасного шаринга экрана или создания скриншотов.
- **Бейджи Ahrefs DR** — карточки сайтов показывают Domain Rating домена, подгружаемый через бесплатный публичный эндпоинт Ahrefs и кэшируемый на 7 дней. *Domain Rating предоставлен [Ahrefs](https://ahrefs.com/).*

### Портфельная SEO-аналитика

Выносите общую SEO-аналитику на главный дашборд OpenGSC:
- **Striking Distance** — ключевые слова из топ-20 на пороге топ-10 по всей сетке сайтов.
- **Каннибализация** — список страниц, конкурирующих за одни и те же запросы, по всей базе доменов.
- **Content Decay** — сводная тепловая карта угасания страниц по всем сайтам для предотвращения просадок трафика.

### Детальная статистика сайта

Каждый домен имеет свою страницу детального анализа с трендами кликов/показов/CTR/позиций и специальными отчетами:
- **Striking Distance Keywords** — запросы на позициях 4–20 с хорошим объемом показов: ваши точки быстрого роста.
- **Keyword Cannibalization** — запросы, по которым конкурируют несколько ваших страниц, с разбивкой на лидера и аутсайдеров.
- **Content Decay Map** — тепловая карта страниц, теряющих клики за последние месяцы.
- **CTR Benchmark** — сравнение реального CTR страниц со стандартными кривыми распределения кликов для поиска страниц с плохим сниппетом.

### Интеграция с Google Analytics 4 и Microsoft Clarity

Связывайте ресурсы GA4 (сессии, вовлеченность, ключевые события, доход) и проекты Microsoft Clarity (записи сессий / тепловые карты за 30 дней) в рамках детальной страницы сайта. Инструкция по настройке: [`docs/GA4-SETUP.md`](docs/GA4-SETUP.md).

### Rank Tracker

Отслеживайте позиции ключевых слов (с привязкой к стране/языку/устройству) через Serper, DataForSEO или ScrapingRobot на одном графике с реальными данными GSC.

### AEO Tracker — видимость в поисковых ИИ-ассистентах

"Answer Engine Optimization": проверяет, **цитируется ли ваш сайт при ответах ИИ-ассистентов** — ChatGPT и Perplexity (через веб-поиск), Claude и Grok (через упоминания бренда), формируя историю упоминаний.

### Проверка бэклинков

База обратных ссылок на сайт с проверкой их доступности и статуса индексации в Google через XML River.

### Проверка здоровья сайта

Панель, объединяющая аудит SSL-сертификата (срок действия, эмитент), проверку на вирусы/фишинг через **Google Safe Browsing** и **VirusTotal**, а также показатели скорости **Core Web Vitals** через PageSpeed Insights API.

### Интеграция с Bing, Яндекс.Вебмастером и IndexNow

- **Переключатель поисковиков на дашборде сайта** — рядом с данными GSC переключаетесь на полноценные живые дашборды Bing (клики, показы, CTR, средняя позиция, график, топ запросов и страниц, страницы в индексе, ошибки обхода) и Яндекса (ИКС, страницы в поиске/исключено, график, топ запросов с позициями, диагностика сайта от самого Яндекса). Кнопка Sync обновляет все подключённые поисковики разом. Инструкция по получению ключей и токенов: [docs/SEARCH-ENGINES-SETUP.md](docs/SEARCH-ENGINES-SETUP.md).
- **Отправка Sitemap** в Bing (API или ping) и Яндекс.Вебмастер в один клик из вкладки Indexing.
- **Переобход URL в Яндексе** — отправка страниц на переиндексацию с учётом дневной квоты, через ваш собственный OAuth-токен.
- **Моментальный пуш URL через IndexNow** — отправка новых и измененных адресов напрямую в поисковые системы (Bing, Yandex) для мгновенного обхода.

### Технический аудит сайта — встроенный краулер

Бесплатный технический аудит без внешних API: OpenGSC сканирует ваш сайт прямо с вашего VPS (до 500 страниц) и сообщает о битых внутренних ссылках, отсутствующих или дублирующихся заголовках title, отсутствующих meta descriptions, проблемах с H1, страницах с `noindex`, несовпадениях канонических адресов, тонком контенте, изображениях без alt-тегов и медленных ответах. Все это сводится в оценку здоровья с фильтруемой таблицей.

### Уведомления и дайджесты в Telegram и Slack

Подключите собственного бота Telegram (настройка через @BotFather за 1 минуту, токен вставляется в **Настройки → Уведомления**) и/или Slack Incoming Webhook — OpenGSC будет присылать отчеты прямо в чат:
- **Алерты** (проверяются каждый час): падение отслеживаемого ключа на N+ позиций, просадка кликов на X% за неделю, скорое истечение SSL-сертификата, низкий балл аудита сайта.
- **Дайджесты** (вкладка **Дайджест**): портфельный отчёт по всем сайтам *или по одному тегу* (например, сетке PBN). Вид **на экране — полный и подробный** (явный диапазон дат, KPI портфеля, сильнее всего выросшие/просевшие сайты, растущие/падающие запросы, *все* запросы на пороге топ-10, сайты, требующие внимания, движения позиций — каждая секция с «показать все» и CSV), а **в Telegram уходит короткая выжимка** (безопасно для лимита в 4 000 символов). **Вкладки Google / Bing / Яндекс** разбивают отчёт по поисковикам (вкладки движков переиспользуют кэш портфеля). Опциональный **AI-вывод** (на вашем AI-ключе) пишется сверху и **включён по умолчанию, если задан хоть один AI-ключ**. Предпросмотр на экране, отправка по кнопке или расписание.

### Гостевые дашборды — read-only ссылки для клиентов

Поделитесь дашбордом сайта с клиентом без создания аккаунта: **сайт → Настройки → Публичная ссылка** генерирует токенизированный read-only URL. Гость видит аналитику только этого сайта (дашборд, позиции, GA4, здоровье, аудит) — без настроек и кнопок действий; ссылку можно перевыпустить или отозвать в любой момент.

### MCP-сервер — подключение ИИ-агентов

В OpenGSC встроен **MCP-сервер** на `/api/mcp`. Любой MCP-совместимый ИИ-клиент (Claude Code, Claude Desktop, Cursor, Codex) может напрямую запрашивать данные о ваших сайтах, эффективности поиска, ключевых словах Striking Distance, каннибализации, отслеживании позиций, аудите, бэклинках, а также выполнять произвольные read-only SQL-запросы к базе данных. Сгенерируйте токен в **Настройки → API и MCP**, затем подключите:

```bash
claude mcp add --transport http opengsc https://your-domain.com/api/mcp \
  --header "Authorization: Bearer <token>"
```

Все инструменты читают локальную базу данных VPS, поэтому запросы ИИ-агентов бесплатны и не расходуют квоты Google/SERP. Подробная инструкция: [`docs/MCP-SETUP.md`](docs/MCP-SETUP.md).

## 🧠 ИИ SEO-пакет инструментов (`/seo-tools`)

Полноценный конвейер от анализа конкурентов до публикации готовых статей. Все модули опциональны и используют ваши собственные API-ключи (Anthropic, OpenAI, Gemini, OpenRouter, Kimi/Moonshot, kie.ai и др.) по себестоимости.

- **Кластеризация ключевых слов**: группировка семантики по схожести выдачи (SERP overlap) на основе реального топ-10 Google.
- **Генератор структуры (Outline)**: парсинг конкурентов по запросу, определение их интента и сущностей, автоматическое построение ТЗ для автора с защитой от ИИ-галлюцинаций (Fact-scrub) и поддержкой Casino RAG для гемблинг-тематик.
- **Генератор текста**: пошаговое написание статьи частями (что сохраняет качество прозы на больших объемах) с контролем объема (Volume guard) и поддержкой линковки на источники.
- **Рерайт контента (Content Rewriter)**: вставьте текст или **URL** (парсится автоматически) → получите **N уникальных вариантов** (1–5) с сохранением смысла, фактов, чисел и ссылок. У каждого варианта — **% уникальности** (сходство по триграммам слов) и число слов, копирование/скачивание. Опции: язык, тон и тумблер **«маскировать AI-паттерны»** (убирает тире, «furthermore», «важно отметить»…). Работает на ваших мульти-провайдерных ключах. Встроен в **Content Decay**: у каждой проседающей страницы — кнопка «Рерайт» с уже подставленным URL. Незаменим для больших аффилиат-сеток, где дубли контента — реальный риск.
- **Googlebot View — детектор клоакинга и PBN**: «увидеть» страницу глазами краулера Google и поймать то, что скрыто от обычного визита. Тянет URL несколькими **User-Agent** (Googlebot mobile/desktop, браузер, опц. Googlebot + `Referer` из выдачи), вручную проходит **цепочку редиректов** и **сравнивает** ответы — вердикт *чисто / подозрительно / клоакинг*, таблица различий (финальный статус/URL, canonical, meta robots, hreflang, JS-редиректы, индексируемость). Для **своих** сайтов в GSC добавляет **True Google View** — реальный вердикт Google через URL Inspection API. Спека: [docs/GOOGLEBOT-VIEW-SPEC.md](docs/GOOGLEBOT-VIEW-SPEC.md).
- **Анализ упущенного контента (Content Gap)**: сравнение вашей страницы с конкурентами в топ-10 и точные рекомендации, чего не хватает на вашем URL.
- **Landing Page Builder**: составление ТЗ, прототипов и текстов для коммерческих страниц. Поддерживает импорт структуры ваших страниц, в том числе по скриншоту.
- **GEO-аудит (Generative Engine Optimization)**: анализ того, как ИИ-поисковики (ChatGPT, Perplexity) цитируют бренды по вашим целевым запросам, и рекомендации по улучшению видимости в ИИ.
- **Трекер упоминаний и тональности**: отслеживание бренда в сети с определением тональности и эмоций через DataForSEO.
- **Мониторинг ссылок (Link Monitor)**: отслеживание свежих качественных ссылок конкурентов через API Ahrefs v3 и поиск сайтов, которые ссылаются сразу на нескольких конкурентов (лучшие доноры для аутрича).
- **Редакционная политика**: создание и применение единых правил стиля, тона и стоп-слов для всех генерируемых текстов.
- **История**: единый серверный лог всех генераций с возможностью возобновления зависших задач.

## 🕸️ Приватная сеть индексации (`/indexer`)

Инструмент управления сетью сайтов-дорвеев для быстрого индексирования целевых страниц.
- **Управление доменами**: создание шаблонов контента (Ecommerce, Блог и др.), настройка редиректов и уникальных ключей.
- **Планировщик перелинковки**: визуальный построитель топологий связей доменов (кольцо, сеть, пирамида) с экспортом HTML-кода.
- **Клоакинг бота**: проверка посетителей по User-Agent и двойному DNS-запросу (PTR-запись). Боты поисковых систем (Google, Yandex, Bing) видят контент дорвея и ссылки для индексации, а обычные пользователи и краулеры-шпионы перенаправляются на целевой сайт.
- Deployed-скрипты доступны в четырех вариантах: Nginx-конфиг, Astro middleware, PHP-дорвей или статический PHP-враппер. Подробно в [`docs/INDEXER-SETUP.md`](docs/INDEXER-SETUP.md).

## Требования к серверу

## 🚀 Установка

Подробный шаг за шагом гайд по настройке Google Cloud Console и запуску скрипта установки описан в английской версии [README.md](README.md#-installation). 

Краткий вариант установки на чистый сервер:
```bash
curl -fsSL https://raw.githubusercontent.com/fenjo26/opengsc/main/install.sh | sudo bash
```
Скрипт спросит домен, установит Node.js, PM2, Nginx, выпустит SSL Let's Encrypt, настроит файрвол и запустит приложение.

Для обновления версии достаточно выполнить:
```bash
cd /root/opengsc
git pull
npm install
npx prisma db push
npm run build
pm2 restart opengsc
```

Полная документация по архитектуре, GA4, Docker, MCP и индексатору доступна в папке [docs/](docs/).
